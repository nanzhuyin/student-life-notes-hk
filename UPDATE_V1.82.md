# Otter v1.82 用户更新说明

发布日期：2026-07-16

## 岭南大学成绩查询指南

新增《岭南大学查成绩：LU Mobile 与 myLingnan Portal 两种入口》，可以在切换到岭南大学后进入“新生入学指导”查看。

本次补充内容包括：

1. LU Mobile 首页点击 `Registry Service` 的第一步图示。
2. `My Academic Result`、`Early Grade Release`、课表、考试时间和毕业要求入口的区别。
3. myLingnan Portal 中通过 `Academic Matters` 查看 `Academic Report` 的网页路径。
4. 正式成绩、提前开放成绩、Academic Report 和正式 Academic Transcript 的区别。
5. VPN、CTLE、学期选择、View Holds 和个人成绩隐私提醒。
6. 可查看的三页 PDF 成绩查询参考节选；与 Otter 硕士范围无关的本科 Cultural Studies 内容没有收入系统。

## 图片与版权说明

网站新增统一的第三方图片提示。用户提供或公开页面中的截图只用于资料核验、操作入口识别和学习参考，相关权利归原作者、学校、发布平台或相应权利人所有。Otter 不主张图片所有权，也不代表学校官方立场。

课程来源截图旁保留原帖入口。权利人如需补充署名、更正来源或移除图片，可通过站内“建议 / 投稿反馈”窗口联系管理员。

详细说明见 `THIRD_PARTY_CONTENT_NOTICE.md`。
